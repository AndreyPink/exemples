# Tbilisi Nanny MVP

## Запуск

```bash
npm install
npm run dev
```

После запуска портал доступен по адресу: http://localhost:3000

Данные анкет и заявок сохраняются локально в `data/store.json`.

## Доступ

Админка: /admin
Пароль: admin123

## Функции

- регистрация нянь
- заявки родителей
- админка
- простой матчинг
